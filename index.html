Git

Git'as yra populiariausia kodo versijavimo programa.

Git Komandos

  • init - sukuria git'o repozitorija
  • add - prideda nurodytus failus prie comit'ui paruostu failu (staged)
  • status - nurodo informacija apie tavo lokalios git'o repozitorijos statusa
  • commit -m "zinute" - sukuria kodo versija is stage'intu failu ir prideda zinute apie pakitimus (zinute bandyti islaikyti ne ilgesne negu 50 simboliu)
  • push - is lokalios repozitorijos commit'u nusiuntimas i isorine repozitorija
  • pull - is isorines repozitorijos commit'u atsisiuntimas i lokalia repozitorija
  • remote add origin https://... - isorines repozitorijos prijungimas prie lokalios repozitorijos
  • log - commit'u sarasas
